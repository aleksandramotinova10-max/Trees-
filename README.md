Заглавие на проекта- Trees

Кратко описание на задачата- Програмата реализира двоично наредено дърво на C# с възможност за добавяне търсене и извеждане на елементи.

Използван програмен език и технологии- език C# като е разработено като конзолно приложение в Visual Studio.

Как се стартира програмата-чрез натискане на Ctrl+F5 можем да стартираме проекта
